# AI 写小说框架搭建及 Prompt 演示

> 简介：AI 不能代替创意，但能极大提速执行。本文提供一套"五步法"写作框架，配合精准 Prompt，让 AI 成为你的 24 小时编辑搭档。

## AI 写小说的核心原则

1. **AI 负责扩写，人负责方向** — 大纲、关键情节转折必须自己定，让 AI 填充细节
2. **一次只让 AI 做一件事** — 不要让 AI 同时构建世界观 + 写第一章，拆分任务
3. **保持上下文一致** — 每次新会话都粘贴"角色设定卡"，防止人物性格漂移
4. **多模型互补** — Claude 擅长逻辑和对话，Kimi 擅长风格模仿，豆包适合快速批量生产

## 工具选型

| 工具 | 优势 | 适合场景 |
|------|------|---------|
| Claude Sonnet | 逻辑最强，对话自然 | 复杂剧情设计、人物心理描写 |
| Kimi（Moonshot） | 中文风格好，上下文长 | 章节扩写、风格模仿 |
| 豆包（Doubao） | 速度快，免费额度多 | 批量生产、大纲头脑风暴 |
| 文心一言 | 适配国内平台风格 | 番茄/起点类型网文 |

## 五步写作框架

### 第一步：世界观设定

```
你是一位资深网络小说编辑，帮我构建一个世界观设定文档。

核心设定：
- 类型：[都市/玄幻/修仙/科幻/言情]
- 主要矛盾：[主角要达成什么目标，面对什么阻碍]
- 世界规则：[力量体系、社会结构、关键背景]
- 独特钩子：[让读者追更的核心设定，例如"主角能看到所有人的死亡倒计时"]

请输出：
1. 世界背景（300字内）
2. 力量体系（等级名称 + 简述）
3. 主要势力（3-5个，各50字）
4. 禁忌/规则（增加戏剧张力的世界约束）
```

### 第二步：人物设定卡

```
基于以下世界观，为主角设计完整人物卡：

世界观：[粘贴第一步输出]

请输出人物卡（保存此内容，每次写作都带上）：
姓名：
年龄/外貌：
核心性格（3个关键词）：
成长背景（100字）：
核心欲望（主角最想要什么）：
核心恐惧（主角最害怕什么）：
口头禅/说话习惯：
弱点/局限（让读者共情的缺陷）：
```

### 第三步：三幕大纲

```
角色卡：[粘贴人物卡]

帮我设计一个[XX万字]网文的三幕式大纲：

第一幕（开头约10%）：
- 钩子（第一章结尾必须留下的悬念）
- 主角现状与触发事件
- 第一个小目标的建立

第二幕（中间约75%）：
- 主角获得能力/资源的关键节点（5-8个）
- 主要反派登场时机
- 高潮前的最低谷（主角几乎失败的时刻）

第三幕（结尾约15%）：
- 终极对决设计
- 主角弧光完成（性格如何成长）
- 结局钩子（是否留续集空间）

每个节点50-100字描述。
```

### 第四步：章节扩写

```
[粘贴人物卡]
[粘贴本章节在大纲中的节点描述]

写作要求：
- 字数：2000-2500字
- 视角：第三人称限知视角，贴近主角内心
- 风格：[起点男频爽文 / 晋江女频甜宠 / 番茄快节奏流]
- 本章必须包含：
  1. 开头300字内给出本章钩子（引发读者继续看的悬念）
  2. 至少一段动作/对话高潮
  3. 结尾必须是悬念或情绪高点，不能平淡收章
- 禁止：大段心理独白、角色说话过于文雅、节奏拖沓的环境描写

请直接输出章节正文，不需要解释。
```

### 第五步：润色与一致性检查

```
请检查以下章节内容并修改：

[粘贴章节内容]

人物卡参考：[粘贴人物卡]

检查项：
1. 角色性格是否与人物卡一致？（重点检查说话方式）
2. 有没有逻辑漏洞？（时间线、能力限制是否矛盾）
3. 节奏是否有过于拖沓的段落？（超过3句的纯景物/心理描写直接标出）
4. 有没有重复用词或重复句式？

输出：
- 修改版正文
- 修改说明（列出改了什么，方便我学习）
```

## 国内平台风格 Prompt 调整

### 起点男频爽文

在扩写 Prompt 末尾加：

```
额外要求（起点男频风格）：
- 开头100字内必须有主角"开挂"时刻或强者姿态
- 对话简洁直接，主角说话有压迫感
- 每章末尾必须有明确升级/获得资源的爽点
- 避免过多情感描写，主角情感戏不超过全章15%
```

### 番茄快节奏流

```
额外要求（番茄快节奏流）：
- 每500字必须有一个小悬念或反转
- 句子短，多用短段落（每段不超过5行）
- 对话占比超过40%
- 主角解决问题要快，不拖沓
```

### 晋江言情甜宠

```
额外要求（晋江女频甜宠）：
- 细节描写男主的外貌/动作，制造心动感
- 甜点要明确：抱、牵手、对视等肢体互动描写细腻
- 主角（女主）要有自己的想法和主见，不要只依赖男主
- 语言风格偏温柔，使用一些比喻增加浪漫感
```

## 工作流工具推荐

- **Notion / Obsidian**：存储世界观文档、人物卡、大纲（写完贴给 AI 的永久参考）
- **Claude Projects**：把人物卡和世界观放进 Project 知识库，每次对话自动携带
- **Excel / 飞书多维表格**：记录每章字数、发布时间、读者反馈
