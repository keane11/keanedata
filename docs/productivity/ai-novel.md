# AI 写中文网文完整指南

> 简介：网文是中文 AI 写作目前回报最高的赛道。本文整合 6 大平台规则、8 大题材完整 Prompt 模板、6 家国产 AI 实测对比、AI 检测对抗、发书与变现全链路，所有 Prompt 与示例输出可直接复制使用。

---

## 1. 中文网文市场全景

不同平台的题材偏好、签约门槛、收益结构差异巨大，先选对平台再动笔。

| 平台 | 主力题材 | 字数节奏 | 签约门槛 | 全勤奖 | 分成比例 | 起步推荐人群 |
|------|---------|---------|----------|--------|---------|-------------|
| **起点中文网** | 男频玄幻/都市/历史 | 单章 3000+，日更 6000–8000 | 3 万字 + 编辑评估 | 月更 6 万字 ¥600，10 万字 ¥1500 | 5:5（VIP 章节）→ 7:3（大神） | 想冲精品/IP 改编 |
| **番茄小说** | 男频系统流/无敌爽文 / 女频先婚后爱 | 单章 1500–2500，日更 1–2 万 | 上传即可，量大优选 | 8 万字 ¥1500（新签作者首月最高 ¥3000） | 流量分账（≈ 千字 ¥1–3） | 量产型新人 |
| **晋江文学城** | 言情/古言/纯爱（耽美） | 单章 3000–5000，日更 4000+ | 实名 + 锁文 + V 章审核 | 无固定全勤，但有"积分榜"奖励 | 千字 ¥0.05/订阅起，VIP 章节 50% 分成 | 女性向写手 |
| **七猫小说** | 男频无敌流/赘婿流 | 单章 1500–2500 | 上传即可 | 月更 6 万字最高 ¥3000 | 流量分账 | 起步快、收益稳 |
| **掌阅/QQ 阅读** | 兼容多题材 | 单章 2500–3500 | 编辑评估 | 月度签约奖 ¥500–1500 | 五五分（含腾讯系联动） | 多平台分发 |
| **SF 轻小说/刺猬猫** | 二次元/同人/无限流 | 单章 3000+ | 编辑评估 | 全勤 ¥500–1000 | 五五分 + 月票奖励 | ACG 圈写手 |

**实用判断**：

- **追求月入 1 万以上 + IP 改编**：起点（玄幻/都市/历史男频）。新书期 30 天爆发力决定全年。
- **追求"打工式稳定收入"**：番茄/七猫，靠量取胜，单本上不去就开新书。
- **追求"安稳深耕女频"**：晋江，订阅基数稳，老书长尾收益强。

> 收益数字来自 2025 年公开作者交流贴均值，实际因签约等级而异。新手建议先在番茄/七猫练手 50 万字，再去起点冲精品。

## 2. AI 写网文的核心原则

1. **AI 负责扩写，人负责方向**：大纲、关键反转、人物弧光必须自己定。AI 只是 24 小时不疲倦的代笔编辑。
2. **一次只让 AI 做一件事**：不要"先生成世界观再写第一章"，每步独立保存，方便重试。
3. **永远带"人物卡 + 世界观卡"**：每次新会话或长对话掉档后，把卡片粘贴回去，性格漂移直接降低 80%。
4. **多模型互补**：DeepSeek-V3 主力扩写（中文质量 + 价格），Kimi 跨章一致性检查（200 万字上下文），Claude 做精修与高潮章节。
5. **降 AI 味是必修课**：番茄/起点/晋江都有 AI 检测，纯 AI 输出会被限流甚至封号，必须手工降味（见第 6 节）。

## 3. 头部题材模板包（8 大题材完整 Prompt + 示例输出）

### 3.1 玄幻修仙（起点男频主力）

```
你是起点中文网资深玄幻编辑，帮我设计一部修仙小说的核心设定。

要求：
1. 力量体系：9 个境界，每境界 3 个小阶，给出每境界的破境关键、能力表现、寿元。
2. 世界结构：3 大势力（正派 / 中立 / 反派宗门），各 50 字介绍。
3. 主角设定：废材逆袭模板，给出"金手指"（独特能力/物品）的获取方式与限制。
4. 第一章钩子：300 字内必须出现"被欺辱 → 觉醒/获得机缘 → 反杀"的爽点弧线。
```

**示例输出（节选）**：

> **境界体系**：练气（凡人寿，30 阶满）→ 筑基（300 年寿，凝结金丹种子）→ 金丹（800 年寿，可御剑飞行）→ 元婴（2000 年，元神显化）→ 化神（5000 年，可主宰一方）→ 合体 → 大乘 → 渡劫 → 仙人。
>
> **金手指**：主角林川被族中天才打入悬崖，因祸得福吞下一枚被前辈遗弃的"焚天古鼎"碎片。古鼎只能在他血滴上炉时激活，每次消耗 100 滴血换 1 次"逆天炼丹"机会（约束：用多了会折寿，迫使他必须谨慎使用 → 制造悬念）。

> 在 Prompt 末尾追加："输出请使用 Markdown 三级标题，方便我直接粘贴到 Notion。"

### 3.2 都市重生（番茄/起点双平台）

```
设计一部都市重生小说。

约束：
- 重生时间点：2008 年 9 月（互联网创业前夜，方便主角"先知"布局）
- 重生原因：前世死于一场资本绞杀，要在前世经历的关键节点上反转
- 金手指：脑海里有"前世记忆 10 年"，但只到 2018 年（强制约束，避免无敌过头）
- 主角性格：表面普通学生，内心冷静、果断、不滥用先知
- 第一章必须发生：用 500 元启动资金 + 一条新闻信息，赚到第一笔 5 万元

请直接输出第一章正文 2500 字。
```

**示例输出片段**：

> 我盯着手机屏幕上「2008 年 9 月 8 日 周一」的日期，喉咙发紧。手心还残留着被那枚子弹擦过的灼烧感——但此刻，我躺在自己 17 岁的宿舍床上，三米外，是闭着眼睛打鼾的老胡。
>
> 三天后，雷曼兄弟倒闭。一周后，A 股从 6124 点跌到 1664 点。这是改写一切的窗口期，而我有十年的记忆做底。

### 3.3 系统流（番茄爆款类型）

```
帮我设计一部"签到+任务"系统流小说。

系统机制：
- 触发方式：完成任务获得"积分"，积分可在系统商城兑换功夫/丹药/装备
- 隐藏关卡：每月 1 号有"隐藏 BOSS"刷新，击败可获得稀有奖励
- 惩罚机制：任务失败扣寿命/属性，制造紧迫感（避免无敌爽文味道太重）

主角设定：
- 22 岁，刚被女友甩，绑定系统
- 第一个任务："在 24 小时内追回前任"——但奖励是"分手分手券"（暗示要拒绝）
- 主角必须在"追回 vs 接受奖励"之间挣扎，性格弧线就从这里展开

输出：系统面板初始化界面（用 ASCII 表格表现）+ 第一章正文 1800 字。
```

**示例输出（节选）**：

```
════════════════════════════════
  ⚡ 任务系统 · 初始化完成 v1.0
════════════════════════════════
  宿主：陈默（22岁 · 男）
  当前状态：失业 · 失恋 · 账户余额 ¥286
  今日任务：24小时内追回前任
  任务奖励：[特殊道具] 分手意志结晶 × 1
  失败惩罚：寿命 -5 年
  ────────────────────────────────
  [点击展开奖励说明]
════════════════════════════════
```

我盯着悬浮在半空中的半透明面板，用了整整三秒判断自己是不是在做梦。

不是。地板还是那块地板，天花板还是那块天花板，微波炉里还有昨晚没吃完的外卖。楼下有人在扔垃圾，袋子撞在桶边上，发出一声闷响。

只是多了一个系统面板。

我点开"奖励说明"。

「分手意志结晶：使用后，主动提出分手时，双方情感损伤降低 87%，对方不会记恨，且后续关系可平稳过渡至普通朋友……」

我把它关掉了。

追她，然后亲手跟她分手。

这个系统在开什么玩笑。

倒计时不开玩笑：**23:47:31**……

我拿起手机，翻到赵雪的头像，按了三秒，删除——停住，重新按回来。任务还有 23 小时 47 分。先去拿外套。

> **注**：系统面板的 ASCII 格式让读者立刻有"游戏感"；奖励道具（分手券）制造悬念——读者想知道主角会不会用它。

### 3.4 历史穿越（起点精品 IP 改编池）

```
设计一部明朝中后期穿越小说。

时空设定：
- 穿越时间：嘉靖三十一年（1552 年），倭寇侵扰浙江
- 穿越身份：戚继光的远房表弟，22 岁秀才
- 主角能力：现代历史系研究生，知道未来 50 年大事件，但不会武功
- 历史尊重原则：不改变重大史实（戚继光必须打赢），主角只能在缝隙中"加速"历史

第一章场景：
- 主角刚穿越，正经历倭寇袭村
- 用"现代知识"想出一个救人方法（不是物理对抗，是组织、声学、心理战）
- 救下 30 个村民，引起当地县令注意

请直接写正文 2500 字，对话使用半文言（"汝/吾/尔/某"），叙述用现代汉语。
```

**示例输出（节选）**：

> 嘉靖三十一年，夏，台州城外。
>
> 倭刀架在我脖子上，我听见自己的呼吸，比预期的要稳。
>
> "汝是何人？！"持刀的少年约莫十六七岁，官兵打扮，左颊有道未愈的刀疤。手在抖，眼神没有——好苗子。
>
> "顾行之，秀才。"我没动，"现在别开口，往后退三丈，蹲下。"
>
> "汝——"
>
> "那片竹林，"我说，声音压得几乎贴着牙关，"火把在移动，倭贼哨兵，至少三人。你没看见？"
>
> 他回头。一秒，两秒，第三秒，他退了。
>
> 我知道他会退的。我知道他叫周勇，活到二十八岁，跟着戚继光打十一场胜仗，名字最终刻进台州府志。但他现在不知道。他只知道祠堂里还有三十二个百姓，和一队人数倍于他们的倭贼。
>
> "尔有何计？"他最终问，声音比我预期的稳。
>
> 我想起研究生二年级那篇讲古代心理战的论文，想起某个军史论坛上关于声学欺骗的帖子，然后——
>
> "你们有多少人会说闽南话？"我问。
>
> 他愣了一下："两人。"
>
> "够了，"我说，"我教你六句话。"

### 3.5 言情甜宠（晋江/番茄女频）

```
晋江女频甜宠小说设定。

人物：
- 女主：26 岁独立设计师，性格清冷理智，不依赖男主
- 男主：30 岁医疗集团继承人，外表禁欲，内心黏人
- 两人关系：先婚后爱（家族联姻），共住一个屋檐下，但分房睡

第一章核心情节：
- 婚礼当晚，女主直接划了"夫妻协议"：分房、AA、不公开秀恩爱
- 男主表面冷静签字，内心暗暗较劲："那就让你心甘情愿撕协议"
- 必须有一个"无意肢体接触"的瞬间（如女主睡着倒在男主肩上），让两人心跳漏拍

风格要求：
- 心理描写细腻，男主视角占 40%，女主视角 60%
- 每 300 字必须有一个"小动作"细节（一个眼神/手指动作/呼吸停顿）
- 禁止："霸道总裁壁咚""莫名其妙强吻"等老套情节

输出第一章正文 3000 字。
```

**示例输出（节选）**：

> 协议签完的那一刻，顾辞川把钢笔放回桌上，没有多看一眼。
>
> 沈若兮把协议书叠好，装进她随身带的牛皮纸袋——旁边放着出门前打好的合同范本和工作日程表。她做事向来如此，把所有可能出错的地方提前钉死。婚姻也是。
>
> "第一条，分房。"她说，"我要最左边那间。"
>
> "好。"
>
> "第二条，AA。"
>
> "好。"
>
> "第三条——"
>
> "不公开。"他接了话，语气平静，像在签一份租房合同，"你不用背完，我看过了。"
>
> 沈若兮停了一下，抬眼。顾辞川正在看手机，西装笔挺，侧脸像被灯削过的线条，没有一处多余。她在想他到底有没有生气——然后打断这个念头。他生不生气，和她没有关系。协议里写清楚了。
>
> "那就这样。"她站起来。
>
> "今晚吃什么？"
>
> "……你问我？"
>
> "厨房是共用的，"他把手机揣回口袋，"如果你不介意，我可以叫外卖。"
>
> 沈若兮看着他一秒，没看出任何言外之意。"随便，我不挑。"
>
> 她拎起牛皮纸袋往楼上走，背后传来他打电话的声音——菜名她没听清，很快又是沉默。
>
> 新婚第一晚，就这样过去了。合理，安全，可控。
>
> 沈若兮躺上床，盯着陌生的天花板，把这三个词在心里复述了一遍，然后翻了个身。

### 3.6 灵异悬疑（中长尾稳定收入）

```
都市灵异悬疑短篇连载。

设定：
- 主角：殡仪馆化妆师，能"看见"死者的最后一刻
- 每个 case 独立成 5–8 章短篇，主线贯穿"为什么主角会有这种能力"
- 单 case 节奏：第 1 章引入死者 → 第 2-3 章追查 → 第 4-5 章反转 → 第 6 章解决 + 留下主线钩子

第一个 case：
- 死者：23 岁外卖小哥，死亡判定为"过劳猝死"
- 主角看到的最后画面：死者在一栋写字楼前抬头，看到 18 楼窗口有人

请输出第 1 章正文 2200 字。

风格要求：
- 第一人称（主角视角），冷静克制，禁止大段惊叹号
- 恐怖感来自"细节错位"，不要直接描写恐怖画面
- 每章末尾必须留一个具体疑问（不是"接下来会发生什么"这种空钩子）
```

**示例输出（节选）**：

> 二十三岁，外卖员，姓赵，叫什么我还没问到。
>
> 他被推进来的时候，脸上是那种长途跑步后的疲倦——嘴微微张着，眼皮没有完全合上，像随时准备被叫醒继续送单。
>
> "过劳。"接班同事扫了一眼病历，"家属还没到，说要等。"
>
> 我没接话，把手套拉到手腕，开始处理。
>
> 死者的手是凉的——这不奇怪。奇怪的是左手食指，指甲断了，断口不像磕的，像是抠过什么，用力抠，抠到最后一刻才停。
>
> 我让自己别多想。工作上的习惯是，只看眼前，不往深处追。
>
> 但等我碰到他手腕，来不及了。
>
> 画面闯进来——
>
> 夜晚，写字楼前。他仰起头，看向十八楼，楼里有盏灯，灯下有个人影，他看见了，对方也看见了。
>
> 然后断了。
>
> 我站在操作台前，等那种头痛的浪退去，然后去洗了手。
>
> 那栋写字楼，就在他出事的路口对面。租的都是律所和会计师事务所，凌晨两点不可能还有人在——但那个影子是真实的。
>
> **他看见了什么，让他死之前要抠断自己的指甲？**

### 3.7 无限流（SF 轻小说/刺猬猫主战场）

```
设计一部无限流小说的主世界设定。

世界规则：
- 主神空间：编号 7726，玩家被"轮回者"系统抽中，每 30 天进入一个副本
- 副本来源：经典恐怖电影 + 现实灾难片，前 5 个副本必须包含 1 个全新原创副本
- 死亡判定：副本内死亡 = 现实死亡（提高紧张感）
- 积分系统：通关获得积分 → 兑换"血脉/功法/物品"，但需付出代价（属性绑定、心魔）

主角设定：
- 28 岁刑警，被一桩冤案逼到自杀边缘时被系统选中
- 进入第一个副本前，先经历"新手村"：在一个"无限循环的早晨"里破解第一个谜题（致敬《土拨鼠之日》）

请输出新手村全章 2800 字。
```

**示例输出（节选）**：

> 早上七点十三分，闹钟响了。
>
> 我按掉它，看见手机锁屏：**2026年3月9日 星期一**。
>
> 起床，刷牙，下楼，买了一根油条一碗豆浆。老板是个四十多岁的女人，围裙上有一块没擦掉的油迹，形状像一只靴子。我在路边吃完，回了单位，在值班室坐下来。
>
> 七点五十四分，卷宗落到我桌上——十六岁少女失踪七十二小时，最后信号在北郊废弃厂房。
>
> 我翻开第一页，看见照片里的女孩——
>
> 然后闹钟又响了。
>
> 我按掉它，看见手机锁屏：**2026年3月9日 星期一**。
>
> 油条，豆浆，围裙上那块油迹。
>
> 第二次，我没去拿卷宗，而是坐下来，从口袋里找出随身带的本子，写下了两个字：**循环**。
>
> 第三次闹钟响起时，我已经在厕所对着镜子，研究自己左手腕上那道细得几乎看不见的印记。像刀痕，又像被什么东西绑过。
>

## 6. AI 检测对抗（必读）

番茄、起点都有 AI 检测系统，命中后轻则限流、降权，重则取消签约。**纯 AI 输出过审率不到 20%。**

### 6.1 检测维度

| 检测维度 | 平台权重 | 典型特征 |
|---------|---------|---------|
| 句式重复 | 高 | "他知道...他明白...他清楚..."同结构连用 |
| AI 高频词 | 中 | "宛如""仿佛""似乎""不由得""微微""轻轻" |
| 段落结构 | 高 | 每段长度过于均匀（AI 默认 3-5 句一段） |
| 信息密度 | 中 | 一句话堆 3 个比喻、4 个形容词 |
| 转折词 | 中 | "然而""不过""但是"高频出现 |
| 情感词堆砌 | 低 | "震惊""恐惧""绝望"直接说出而不展现 |

### 6.2 降 AI 味 7 招

1. **打散段落节奏**：故意让某段只 1 句话，下段 8 句话，模拟人类思绪。
2. **删掉 60% 的"微微/轻轻/不由得"**：这些是 AI 最爱的填充词。
3. **加入口语化的"破句"**：『他不想听。真的不想。一个字都不想。』
4. **加入"无意义但真实"的细节**：『他从口袋里摸出半包烟，发现还剩一根。点上，发现打火机没气了。』
5. **替换 AI 高频比喻**：把"宛如一只受惊的小鹿"改成自定义的、和人物背景相关的比喻。
6. **手改对话**：AI 写的对话过于"完整"，人话有省略、有口头禅、有打断。
7. **检测工具**：用「朱雀 AI 检测」「GPTZero」自检，目标 AI 概率 < 30%。

### 6.3 实战流程

```
AI 初稿 → 朱雀检测（看 AI 概率）→ 手改 60% 句子 → 二次检测 → 概率 < 30% 上传
```

| 平台 | 第一指标 | 健康值 | 红线（低于则换书） |
|------|---------|--------|------------------|
| 起点 | 收藏数 + 推荐票 | 上架前 3000 收藏 | 1500 收藏以下 |

| 晋江 | 收藏数 + 长评数 | 5000 收藏 + 30 长评/周 | 2000 收藏以下 |

### 起点新书期 30 天打榜策略

```
Day 1-3：日更 8000 字 + 求收藏（每章末尾写一句"求个收藏，下章见"）
Day 4-7：日更 6000 字 + 求推荐票
Day 8-14：开始铺核心爽点，争取上"新书强推"
Day 15-21：维持日更，参加月榜
Day 22-30：冲签约 / 冲首页推荐
```

### 留存率诊断

- 第 5 章流失 > 50%：开篇钩子失败，重写前 5 章
- 第 50 章流失 > 50%：第一个高潮拉胯，调整剧情节奏
- 第 100 章后流失：金手指逻辑出问题或反派太弱

## 10. 严肃文学 / 出版小说参考

如果目标不是网文流量收入，而是想写"能放到书架上的小说"或冲文学奖、出版社，思路完全不同。

### 10.1 与网文的核心差异

| 维度 | 网文 | 严肃/出版小说 |
|------|------|--------------|
| 节奏 | 每章必爽点 | 全本一个核心问题 |
| 语言 | 直白清晰 | 留白、潜台词、意象 |
| 人物 | 主角必须越来越强 | 主角必须有"不可挽回的失败" |
| 钩子 | 每章一个 | 全书一个 |
| 字数 | 100 万 +  | 10–25 万 |

### 10.2 Prompt 风格调整

```
将以下网文风格段落改写为严肃文学风格：

原文：[粘贴网文段落]

改写要求：
- 禁止形容词堆砌（每句最多 1 个形容词）
- 禁止"爽点"（不要让主角立即赢、立即成长）
- 引入"留白"：主角的情感不要直说，用动作/物品/天气暗示
- 引入"意象"：找一个具象物品（如一把旧伞、一个搪瓷杯），让它在场景中反复出现，承载情感
- 句长有节奏：长短句交错，至少有一句极短句（≤ 3 字）

输出改写版 + 改写思路说明。
```

### 10.3 同一大纲两种写法对比

> **大纲**：父亲过世后，主角回老家收拾遗物，发现父亲留下一封从未寄出的信，写给一个主角从未听说过的女人。

**网文版（开篇）**：

> 林溪推开吱呀作响的木门，灰尘扑面而来。他在父亲的旧书柜里翻找时，一封泛黄的信笺从《唐诗三百首》里掉了出来。他捡起信，一行隽秀的字迹映入眼帘：
> 「致我永远的兰心：」
> 林溪手一抖。父亲一辈子没提过这个名字。

**严肃文学版（开篇）**：

> 木门是开着的。我没有推。
> 屋子里只有钟。
> 父亲的书柜第三层，那本《唐诗三百首》压在底下，封面发软。书签是一片银杏叶，干透了，捏一下就碎。我没捏。我把书翻开，从中间。一张信纸滑出来，落在地板上的姿势，像他最后那次摔倒。
> 信开头是"致我永远的兰心"。
> 我蹲下去，没去捡。

### 10.4 出版/投稿路径

| 渠道 | 适合作品 | 流程 |
|------|---------|------|
| 豆瓣阅读「征文大赛」 | 中短篇（5–15 万字） | 投稿 → 编辑评估 → 电子书上架 + 比赛奖金 |
| ONE·一个 / 故事 FM | 短篇（5000-15000 字） | 投递邮箱 → 收入约 ¥500-3000/篇 |
| 《人民文学》《收获》 | 纯文学短篇/中篇 | 邮件投稿，回复周期 3-6 月，难度极高 |
| 后浪 / 新经典 / 单向街 | 长篇出版 | 通常需要文学奖背书 / 已有读者基础 |
| **Kindle Direct Publishing** | 中长篇，无需出版社 | 30 分钟即可上架，70% 版税 |
| 微信读书原创 | 中长篇 | 签约后按阅读时长付费，门槛较低 |

## 12. 第一章开篇技法

第一章是决定留存率的关键。平台算法会统计"第一章完读率"——低于 40% 的新书几乎不会进推荐池。

### 12.1 五种开篇模式

| 模式 | 适用题材 | 核心逻辑 | 第一句示例 |
|------|---------|---------|-----------|
| **危机开场** | 玄幻、系统流 | 直接把主角扔进绝境 | 『渡劫最后一个时辰，陈默发现金丹碎了。』 |
| **金手指展示** | 重生、穿越 | 立刻让读者看到"我有外挂" | 『林秋重新睁眼，手机屏幕显示：任务完成，奖励：2047 年的股价表。』 |
| **悬疑开场** | 灵异、无限流 | 提出一个无法解释的反常 | 『这是他第 17 次在同一天醒来，尸体还是那具，凶手还是不知道。』 |
| **强情绪锚定** | 言情、甜宠 | 用极具冲击力的情绪瞬间抓住读者 | 『婚礼当天，新郎叫了另一个女人的名字。』 |
| **反差笑点** | 搞笑系统流、轻松向 | 设置荒诞反差，引发好奇 | 『系统给我的第一个任务是：让宿敌爱上我——对象是一只猫。』 |

### 12.2 开篇钩子公式

**有效开篇 = 一个事实（异常）+ 一个问题（悬念）+ 一个情绪（代入）**

> 示例（玄幻）：『他记得，自己七岁时已经死过一次。（事实异常）所有人都说他是侥幸，但那个声音说的是：下次别这么早死。（悬念）他一直以为那是梦。直到今天，他再次躺在血里，那个声音又来了。（情绪代入）』

### 12.3 第一章专用 Prompt

```
为以下网文写第一章（2000-2500 字）：

题材：[题材]
主角：[姓名 + 核心设定一句话]
开篇模式：[危机开场 / 金手指展示 / 悬疑开场 / 强情绪锚定 / 反差笑点]
第一个爽点/钩子：[具体内容]
第一章必须包含：
1. 前 200 字内给出"这不是普通故事"的信号
2. 主角的一个标志性行为（后续可以反复呼应）
3. 一个没有解决的悬念，作为第二章驱动力
4. 结尾一句话必须让读者想立刻点"下一章"

禁止：
- 主角有大段内心独白介绍背景
- 出现"这一切还要从三年前说起"类回忆
- 前 500 字没有实质性事件
```

**第一章质检清单（发布前自查）**：

- [ ] 第 1 句话是否有画面感？
- [ ] 前 300 字内是否出现主角姓名？
- [ ] 第一个冲突/事件是否在 500 字内发生？
- [ ] 读完第一章，读者是否有一个迫切想知道答案的问题？
- [ ] 结尾是否有"悬而未决"的钩子？

## 13. 配角与反派设计

网文失败 60% 不是因为主角弱，而是配角脸谱化、反派太蠢。

### 13.1 对照映射技术

每个核心配角/反派都应该是主角的"镜子"——展现不同选择下的另一种可能性。

| 类型 | 主角属性 | 对应配角/反派 | 对照意义 |
|------|---------|------------|---------|
| **黑化镜** | 底层逆袭，但保持善良 | 同样底层逆袭，但抛弃底线 | "如果我做错一步选择…" |
| **能力镜** | 靠努力/智慧成长 | 天才型，但停滞不前 | "天赋不是终点" |
| **信念镜** | 为集体/家人牺牲 | 只为自己，效率更高 | 质疑主角价值观 |
| **阶级镜** | 底层出身 | 世家贵族，待遇天差地别 | 制造仇恨值+社会批判 |

### 13.2 反派塑造 Prompt

```
为以下小说设计主要反派：

主角核心特质：[如"草根出身、重情重义、极限逆袭"]
反派设计方向：[黑化镜 / 能力镜 / 信念镜 / 阶级镜]
反派基本背景：[一句话描述]

请给出：
1. 反派的童年创伤（必须与主角的某个信念直接冲突）
2. 反派的"合理性"：他的逻辑是什么？（读者要能理解，但不认同）
3. 反派的标志性行为/口头禅（3 条，每次出场必体现其中之一）
4. 反派与主角第一次正面冲突的场景（200 字）
5. 反派的"高光瞬间"（他曾经做过一件主角也佩服的事，300 字）
6. 最终大战时，反派说给主角的一句话（不是嚣张语录，而是让主角动摇的话）
```

### 13.3 工具人配角升级

将 "A 出现 → 帮主角 → 消失" 的工具人升级为立体配角：

```
以下配角只是工具人，请帮我丰富：

配角：[姓名 + 现有设定]
他/她在书中的主要功能：[如"给主角提供装备"]

请补充：
- 他/她自己的目标（与主角目标不完全一致）
- 他/她不支持主角的一件事（及原因）
- 他/她的私下生活（主角不在场时他/她在做什么）
- 100 字的配角专属场景（只写他/她，不出现主角）
```

## 14. 卡文突破方案

卡文（写不下去）99% 是因为前面的设定造成了"逻辑死锁"。不要硬撑——分析卡点类型，针对性解决。

### 14.1 五种卡文类型与解法

| 卡文类型 | 诊断标志 | 解法 |
|---------|---------|------|
| **爽点枯竭** | 主角已经"无敌了"，不知道下一个打谁 | 重置对手等级、引入新维度矛盾（从武力 → 政治/情感/身份） |
| **逻辑死锁** | 前面设定互相矛盾，写什么都对不上 | 用"隐藏信息"补丁：某个角色一直在撒谎 |
| **情感平原** | 主角和配角关系没有张力，对话没意思 | 强制引入"第三方"制造三角张力 |
| **节奏塌方** | 刚结束一个大高潮，不知道怎么进入下一段 | 写一章"日常垫场"，用细节积累下一个高潮的情绪能量 |
| **目标漂移** | 写着写着忘记主角要做什么了 | 回顾人物卡和大纲，重新确认"最近一个短期目标" |

### 14.2 卡文急救 Prompt

```
我的小说卡文了，请帮我分析和突破：

当前情况：
- 已写字数：[X 万字]
- 上一章发生的事：[100 字总结]
- 我卡在：[具体描述，如"主角刚赢了大战，不知道下一步写什么"]

我的主角目标（全书）：[一句话]
我的主角目标（当前阶段）：[一句话]

请给出：
1. 我卡文的根本原因诊断
2. 3 条突破方向（每条给出具体的"下一章第一句话"）
3. 推荐方向 + 理由
4. 如果我选择推荐方向，接下来 3 章的简要节点（每章 20 字）
```

### 14.3 灵感刺激 Prompt（换视角）

当卡文无法分析原因时，用换视角强制推进：

```
以下是我的小说当前节点：

[粘贴最近 500 字]

请用以下三个不同人物的视角，各写 200 字描述此刻他们在想什么：
1. 主角的敌人（此刻他在哪里、在想什么）
2. 主角最在意的人（此刻他们在做什么）
3. 一个无关路人（见证了这个场景的任意旁观者）

写完后，告诉我：哪个视角最让你（AI）感到"这里还有故事可以挖"？
```

## 15. 读者运营与评论区策略

很多作者只关注写稿，忽视读者运营。但平台算法会统计"互动率"——评论区活跃的书推荐权重更高。

### 15.1 三平台互动差异

| 平台 | 核心互动指标 | 读者特征 | 运营重点 |
|------|------------|---------|---------|
| 起点 | 推荐票 + 月票 + 打赏 | 有付费习惯，愿意长期追更 | 每章末尾一句"求票"，月底冲榜 |
| 番茄 | 7 日追读率 + 完读率 | 习惯免费，流失快 | 每章字数控制在 2000-3000 字（小白读者怕长） |
| 晋江 | 收藏 + 长评 + 地雷（打赏） | 女频受众，情感细腻，社区文化强 | 回复每条长评，用作者有话说与读者互动 |

### 15.2 评论区运营 SOP

**起点/番茄 评论区**：

```
每日互动清单：
□ 回复 3-5 条读者评论（优先回复有内容的评论，不只是"谢谢"）
□ 在章节末尾预告下章内容（引发讨论，如"明天见秦王，猜猜主角怎么应对"）
□ 每 5 章发一次"作者有话说"：解释一个读者看不到的设定细节
□ 每月举行一次"读者起名/命名"活动（一个次要NPC让读者命名）
```

**晋江读者运营**：

```
晋江长评回复模板（让读者感受到被认真对待）：

读者评论了：[粘贴评论内容]

请给出：
- 一段 100-150 字的作者回复
- 要体现：①确认读者的解读是否接近作者意图 ②透露一个下文的小暗示（但不剧透） ③表达作者写这段时的情感状态
- 语气：真诚、有温度，像和老朋友聊天，不要官方感
```

### 15.3 数据分析 → 内容调整

每周末用 10 分钟做一次简单复盘：

```
本周数据对比（填入实际数据）：
- 均订（平均订阅）：[本周] vs [上周]
- 追读率：[本周] vs [上周]
- 评论数：[本周] vs [上周]
- 高互动章（评论最多）：第 [X] 章，内容是 [X]
- 低互动章（评论最少）：第 [X] 章，内容是 [X]

分析：
高互动章与低互动章最大的差异是什么？
（例：读者喜欢看主角和 A 角的互动，不喜欢纯战斗场景）

下周计划：多写 [X] 类场景，减少 [Y] 类场景。
```
