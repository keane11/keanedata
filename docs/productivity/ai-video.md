# AI 短剧 / 视频生成教程

> 简介：AI 视频生成已经从玩具变成实用工具。本文覆盖国内可用的主流平台对比、制作全流程，以及保持角色一致性的关键技巧。

## 主流工具对比（2025 年）

| 工具 | 优势 | 局限 | 免费额度 |
|------|------|------|---------|
| **快手可灵（Kling）** | 运动流畅，写实风格强 | 创意风格较弱 | 每日 66 积分 |
| **海螺 AI（Hailuo）** | 人物一致性最佳，面部稳定 | 运镜较固定 | 每日有限次数 |
| **即梦 AI（Jimeng）** | 字节系，剪映生态打通 | 时长较短 | 每日免费次数 |
| **Vidu** | 镜头感强，电影感好 | 国内访问有时不稳 | 新用户有额度 |
| **Runway Gen-3** | 国际标杆，运动最自然 | 需翻墙，较贵 | 免费 125 积分 |
| **Sora** | OpenAI 出品，叙事理解强 | 排队严重，限量 | 仅 Plus/Pro 用户 |

**国内首选：可灵 + 海螺组合**（可灵做动态场景，海螺保持人物面部一致）

## 制作全流程

```
剧本创作 → 分镜设计 → 角色参考图生成 → 视频片段生成 → 配音/字幕 → 剪辑合成
```

### 第一步：剧本创作（AI 辅助）

短剧剧本结构：**3 分钟 = 9 个场景，每场景 20 秒**

用 Claude 快速生成剧本框架：

```
帮我写一个3分钟短剧剧本，类型：[都市/古装/科幻]。

要求：
- 共9个场景，每场景描述控制在50字内（包含：画面内容、角色动作、关键台词）
- 第1场景必须有强钩子（让人继续看的悬念或冲突）
- 第5场景是情绪最高点
- 第9场景是反转或升华收尾
- 台词总量不超过50句，每句不超过15字

输出格式：
场景1：[画面描述] / [人物动作] / 台词："..."
```

### 第二步：分镜与提示词设计

每个场景转化为视频生成提示词（以可灵为例）：

```
[镜头类型] + [主体描述] + [动作/表情] + [场景环境] + [光线/氛围] + [运镜方式]

示例：
"近景，25岁东方女性，黑色长发，微微回眸，嘴角带着神秘微笑，
背景是夜晚的霓虹街道，雨后地面反光，镜头缓慢推近，电影感"
```

### 第三步：保持角色一致性

**最关键环节**，失败的 AI 短剧 80% 败在角色面部不一致。

**方法一（推荐）：参考图锁定**
1. 用 Midjourney / 即梦文生图生成角色正面参考图
2. 可灵/海螺支持"图生视频"，上传参考图后角色外形相对固定
3. 每次生成都带上同一张参考图

**方法二：角色提示词模板**

建立角色描述词库，每个镜头都加上：

```
[角色标准描述]：25岁中国女性，杏眼双眸，高鼻梁，樱桃唇，
黑色直发及腰，穿浅蓝色旗袍，气质清冷
```

**方法三（海螺专属）：** 海螺 AI 支持"角色一致性"功能，上传 2-3 张参考图后，系统自动维持面部特征。

### 第四步：配音与字幕

**免费工具：**
- **剪映**：AI 配音 + 自动字幕，国语/粤语/方言均支持
- **海螺 AI**：可直接在平台内添加 AI 配音

**配音 Prompt（给 AI 写台词时配合）：**

```
台词要求：
- 每句台词不超过15字，便于配音节奏
- 情感标注：[哭泣]、[愤怒]、[轻笑]（给配音AI参考）
- 停顿位置用"..."标明
```

### 第五步：剪辑合成

推荐工具：**剪映**（国内最易用）

```
1. 导入所有 AI 生成片段
2. 按分镜顺序排列
3. 添加转场（溶解/推拉效果最自然）
4. 导入配音并对齐口型
5. 添加字幕（剪映"智能字幕"一键生成）
6. 调色（应用统一 LUT，保持风格一致）
7. 导出 1080p，码率 8 Mbps 以上
```

## 常见问题

**Q: 角色嘴型和配音对不上？**

A: 可灵/海螺生成时勾选"嘴型同步"，或用剪映的"智能对齐"功能微调。

**Q: 生成的视频有明显 AI 痕迹（抖动、变形）？**

A: 降低运动幅度，提示词加"稳定的镜头，轻微运动"；可灵选"低运动幅度"。

**Q: 多个场景光线风格不统一？**

A: 每个提示词末尾加统一的风格描述词，如"电影级光线，柔光，冷色调"。
