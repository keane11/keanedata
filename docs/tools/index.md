# 工具箱

一些日常开发和工作中常用的小工具，全部在浏览器本地运行，数据不上传。

## 开发工具

| 工具 | 功能 |
|------|------|
| [QR 码生成器](./qrcode) | 文字或链接 → QR 码图片，支持下载 PNG |
| [Hash 计算器](./hash) | SHA-1 / SHA-256 / SHA-512，支持双文本对比 |
| [Base64 编解码](./base64) | 文本/文件 Base64 互转 |
| [JSON 格式化](./json) | JSON 格式化、压缩、验证 |
| [时间戳转换](./timestamp) | Unix 时间戳 ↔ 日期时间互转，实时时钟 |

## AI 辅助

| 工具 | 功能 |
|------|------|
| [Prompt 构建器](./prompt) | 结构化 Prompt 生成，内置代码/翻译/写作/分析四种模板 |

## 服务

| 工具 | 功能 |
|------|------|
| [文件共享](./share) | QR 码文件共享，三级权限、密码保护、自动过期 |
